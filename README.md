# system-analize
My software for making homework from System Analysis

### Instructions:

1) Clone the repo to your local machine
2) Make sure you have nodejs installed: https://nodejs.org
3) Go to the root of the project
4) "npm install" command to install required dependencies
5) In the file "index.js" enter your matrix in const "MATRIX" and save
6) "npm start" to start the program
